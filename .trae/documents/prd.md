# 招贝产品演示视频页面 - 产品需求文档

## 1. 产品概述

「招贝」AI 亲子财务规划工具的产品演示视频页面，用于比赛展示。
- 通过浏览器自动播放一段约 165 秒的横屏产品演示动画
- 风格接近 Apple Keynote 产品演示片 + Calm 情绪片 + iOS 高保真 App Demo
- 用户可通过浏览器录屏或 Remotion 导出 1080p MP4 视频

## 2. 核心功能

### 2.1 视频章节结构
| 章节 | 时间 | 内容 |
|------|------|------|
| 问题开场 | 0-15s | 账单卡片飞入堆叠，表现家长焦虑 |
| 品牌出现 | 15-30s | Logo 生成，品牌标语 |
| 智能体人设 | 30-50s | AI 智能体名片展示 |
| 服务流程 | 50-72s | 7步流程图展开 |
| Dashboard | 72-98s | App 首页驾驶舱 |
| 支出管理 | 98-120s | 焦虑消费识别 |
| 教育金规划 | 120-140s | 路径模拟与目标管理 |
| AI 顾问 | 140-153s | 结构化决策方案 |
| 成长账本 | 153-162s | 支出关联成长事件 |
| AI 不可替代性 | 162-170s | 四大能力壁垒 |
| 品牌收束 | 170-180s | 收尾与品牌强化 |

### 2.2 播放控制
- 播放/暂停/重播按钮
- 进度条与时间显示
- 自动播放支持
- 隐藏控制栏选项

### 2.3 字幕系统
- 底部旁白字幕同步显示
- 字幕数据集中配置

## 4. 用户界面设计

### 4.1 设计风格
- **整体风格**: Apple Card / Calm / Notion / iOS 高保真
- **金融专业，但亲子温暖**
- **颜色系统**:
  - 暖白背景: #FFF9F5
  - 浅珊瑚: #FFF0E6
  - 珊瑚橙: #FF8C69
  - 薄荷青: #4ECDC4
  - 雾蓝: #5B8DEF
  - 招行红: #E60012（仅核心 CTA）
  - 深色开场背景: #121826
  - 主文字: #172033
  - 次级文字: #6B7280
- **UI 特征**: 大圆角卡片、轻阴影、半透明玻璃拟态少量使用
- **字体**: 中文使用系统字体栈，英文使用 SF Pro / system-ui

### 4.2 页面设计
- 16:9 横屏比例
- 基准分辨率 1920x1080
- 所有文字清晰可读
- 右下角极简进度条
- 底部旁白字幕区域

### 4.3 动画风格
- 柔和、克制、有高级感
- 卡片 stagger 入场
- 数字 count-up 动画
- 进度条平滑增长
- 页面转场柔和
